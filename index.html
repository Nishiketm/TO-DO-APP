FEATURES TO IMPLEMENT

Add new task

Mark task as completed / pending

Edit a task

Delete a task

Filter: All / Pending / Completed

Show total, pending & completed count

Save tasks in localStorage

Dark / Light mode toggle (optional)

Responsive for mobile & desktop

DESIGN GUIDE

  • Dark theme with purple primary color
  • Modern, clean and minimal UI
  • Smooth hover effects on buttons and icons
  • Use cards / shadows for depth
  • Fully responsive (mobile first)
  • Use animations for adding/removing
    tasks (optional but recommended)

TIPS

  • Focus on functionality first, design polish later
  • Break your JS into small functions
  • Keep your code clean and well-structured
  • Test all features before moving to next
  • Use meaningful variable names
Build it. Break it. Fix it. That's how you learn and level up
